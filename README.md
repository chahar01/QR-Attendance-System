# QR Code Based Attendance Management System

## Overview

QR Code Based Attendance Management System is a Java and MySQL-based application designed to automate student attendance tracking. The system generates unique QR codes for attendance sessions and records attendance data in real time.

## Features

### QR Code Generation

* Dynamic QR code generation
* Session-based attendance management

### Attendance Recording

* Scan QR codes to mark attendance
* Real-time attendance updates

### Student Database Management

* Store and manage student records
* MySQL database integration

### Reporting

* Generate attendance reports
* Export attendance records to CSV/Excel

### Validation

* Basic attendance validation mechanisms
* Reduced proxy attendance attempts

## Technologies Used

* Java
* MySQL
* HTML
* CSS
* JDBC
* ZXing Library


### Configure Database

1. Create MySQL database.
2. Import SQL schema.
3. Update database credentials.

### Run Project

Open project in NetBeans or Eclipse and run the application.
